Tutorial to have a better usage experience

Step 1: You need to upload a file that represent the structure tree
Rules about format file:
  1. Format file content:
    • The file must be ".txt" extension
    • The file uploaded must only contains numbers and "N" separate by commas
    • "N"represent null references
    • Nodes values should be between the range[-99,99]
    • The sequence of numbers must follow level order tree's structure
    • Example: [1,2,3,N,5]
Step 2: Choose traversal algorithm you want to perform on tree
  1. Click on Tree traversal algorithms button then it will show the following algorimths:
    • Inorder
    • Preorder
    • Postorder
    • Breadth-First search
  2. Select one of them
Step 3: Learn and enjoy with animation!
Inorder Preorder Postorder Breadth-First Search
Inorder Algorithm (Tree)
  1. Traverse the left subtree
  2. Visit the root
  3. Traverse the right subtree
Preorder Algorithm (Tree)
  1. Visit the root
  2. Traverse the left subtree
  3. Traverse the right subtree
Postorder Algorithm (Tree)
  1. Traverse the left subtree
  2. Traverse the right subtree
  3. Visit the root
